Waste Entry & Logging

Staff log food waste entries by weight, food type, location, and date — plus full sensor readings including pH, temperature, humidity, gas levels, and bin fullness.

Live Staff Dashboard

Bulk CSV Import

Student Dashboard

Reports & CSV Export

Filter waste data by date range, location, or published status — then export to CSV or print a formatted report ready for sustainability meetings and audits.

Campus Leaderboard

Sensor & Bin Monitoring

Track pH, temperature, humidity, methane gas, and bin fullness per entry. The staff dashboard flags any bin above 80% fullness with a live alert banner.

Location Heatmap

Sustainability Tips Feed

Role-Based Access Control

MunchiesAI

The AI assistant that knows your campus data

MunchiesAI doesn't give generic advice. Every response is grounded in live numbers pulled from the database the moment you ask.

Live Data Context

Before every response, MunchiesAI reads your actual campus numbers — total waste, compost, diversion rate, top locations, and week-over-week trends.

Week-over-Week Analysis

Automatically compares this week against last week and tells you whether waste is improving, worsening, or flat — and what to focus on next.

Location Intelligence

Knows which locations generate the most waste and which have the best or worst diversion rates — so it can give location-specific recommendations.

Plain English Questions

No dashboards to dig through. Just ask: "What is our diversion rate?", "Which bin needs pickup?", or "Give me 3 tips to reduce waste this week."

Dual AI Provider

Powered by Claude (Anthropic) with automatic fallback to Llama 3 via Groq — so MunchiesAI stays available even during provider outages.

Auth-Protected

MunchiesAI is only accessible to signed-in users. Rate-limited to 40 messages per 15 minutes to keep usage fair across the campus community.

Bulk CSV Import

Upload a week of data in seconds

Instead of entering entries one by one, staff can prepare a CSV file off-system and upload the whole batch at once.

Drag-and-drop upload

Drop a .csv file or click to browse. The importer reads it instantly in the browser — no server round-trip until you approve.

Row-by-row validation preview

Every row is checked for required fields and valid formats before import. A colour-coded preview table shows exactly which rows will succeed or be skipped.

Blank template download

Download a pre-formatted CSV template with example rows and column headers — so the format is always correct the first time.

Live progress bar

Rows are submitted one by one with a real-time progress bar — so you always know how far along the import is and how many succeeded.
