CSV Format Guide

Required column headers
First row must be column headers exactly as shown below. Order doesn't matter.
foodTypee.g., Leftover pasta
weightlbs, e.g., 4.5
dateYYYY-MM-DD
locatione.g., Dining Hall #1
temperature°F, default 70
humidity%, default 50
ph0–14, default 7
gasppm, default 0
binFullness%, default 50
compostWeightlbs, default 0
notesOptional text

* Required fields. All others use the shown defaults if omitted.
